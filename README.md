# What Is This?

This Is A **Grand Theft Auto: San Andreas** Modloader

## Features
* **Single File**: If Your Game Don't Have **virbosFile.dll** It **Create virbosFileHook.dll**

## Requirements
* **i686-w64-mingw32-gcc** (mingw-64)
* **make** 

## How To Install?

In The Source Dir Run Command

```
make
```
